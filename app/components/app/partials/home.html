Thank you for using the Monitor Local Beta, we hope you will help us to make a great product by providing feedback

What’s missing

While Monitor Local is now in Beta, it is still software in active development and you are likely to encounter some issues or notice missing functionality during this phase. We will be updating the software during the beta phase to fix issues and add functionality.

There are several areas where we know there are issues - for example:

  • Compliance checks: Currently only compliance checks for RCUK have been implemented, and in some cases compliance checks display for non-RCUK funded items
  • All “Manage” screens: There are currently very limited (or no) search options on the Manage screens. We are working on a fully featured search function which will be released with a future software update
  • Integration with external systems: We are still working on the integration between Monitor Local and other systems such as Sherpa, Lantern and Monitor UK
This is not a comprehensive list but may help ensure you don’t report issues that we are already aware of.

If you encounter major issues with the system...

If you encounter an issue that stops you using the system effectively (such as being unable to login, unable to enter or view information, on screen errors etc.) then please report this by email to owen@ostephens.com. We will try to fix any such errors as quickly as possible so you are able to get on with using the system.

If you have feedback on missing functionality or suggested enhancements

Feedback on missing functionality or ways in which we could improve the system should be reported via the form at http://goo.gl/forms/dC06E0vRMT4rRCVf2

The form will prompt you for various information including what screen it relates to and what improvement you would like to see. We’ll be gathering and analysing all feedback during the beta testing and either fixing the issue (e.g. where we are missing functionality needed for the system to work effectively) or adding to the list of enhancements that could be added to the system following the initial launch.

Get Started

The URL for Monitor Local beta is https://monitor.jisc.ac.uk/. To use the system you will need to login and set your user profile information. This takes a few steps, and you will need to read the instructions available at http://bit.ly/29yKF79.
